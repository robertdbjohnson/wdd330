        Here are a few basic animations that might be helpful to reference:

        @keyframes moveRight { 
            from {
            transform: translateX(-50%);
            }
            to {
            transform: translateX(50%);
            }
        }

        @keyframes appearDisappear { 
            0%, 100% {
            opacity: 0;
            }
            20%, 80% {
            opacity: 1;
            }
        }

        @keyframes bgMove {
            100% {
            background-position: 120% 0;
            }
        }

        Here is some shorthand animation properties:
        .verbose {
            animation-name: appearDisappear; 
            animation-duration: 300ms; 
            animation-timing-function: ease-in;
            animation-iteration-count: 1;
            animation-direction: alternate;
            animation-delay: 5s;
            animation-fill-mode: backwards;
            animation-play-state: running;
        }
        /* shorthand */
        .concise {
            animation: 300ms ease-in alternate 5s backwards appearDisappear;
        }
